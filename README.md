# To-Do App (JavaScript)

## 📌 Description
A simple To-Do application built using HTML, CSS, and JavaScript.  
Users can add, delete, mark tasks as completed, and clear all tasks.

## 🚀 Features
- Add tasks
- Delete tasks
- Mark tasks as completed
- Clear all tasks

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript (DOM manipulation)

## 🎯 Learning Outcome
- Learned DOM manipulation
- Event handling
- Building interactive UI
